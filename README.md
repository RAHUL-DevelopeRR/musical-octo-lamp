# musical-octo-lamp
This project is meant to be the Actual Realtime Streaming Engine of the VLC Player, Generating Offline AI Subtitles

## Submodules

This repository includes the [VLC Media Player](https://github.com/videolan/vlc) source code as a git submodule under the `vlc/` directory.

To clone this repository along with the VLC submodule, run:

```bash
git clone --recurse-submodules https://github.com/RAHUL-DevelopeRR/musical-octo-lamp.git
```

If you have already cloned the repository without the submodule, initialize it with:

```bash
git submodule update --init --recursive
```
